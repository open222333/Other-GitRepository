# Other-GitRepository

```
執行後自動拉取 指定資料夾內所有git專案資料夾
```

# 用法

## 配置文檔

```ini
; 複製樣本檔 git_config.ini.default 建立 git_config.ini
; 處理資料夾位置
DIR_PATH=
; git使用者
USERNAME=
; git token
PASSWORD=
; git主機 預設 github.com
GITDOMAIN=
; 設定紀錄log等級 DEBUG,INFO,WARNING,ERROR,CRITICAL 預設WARNING
; LOG_LEVEL=
```

## 終端機執行

```bash
python pull_all.py
```
